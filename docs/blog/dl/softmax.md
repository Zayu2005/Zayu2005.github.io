# Softmax

Softmax是一种常见的激活函数，主要是用于多分类问题中，将原始输出转化为概率分布。核心作用是确定输出值在0-1之间且和为1。

## 公式定义

Softmax函数对于输入向量 $(Z_1,Z_2,...,Z_k)$的第 i个输出为：
$$\left(e^{z_{i}}\right)\sigma(\mathbf{z})_{i}=\frac{e^{z_{i}}}{\sum_{j=1}^{K} e^{z_{j}}}$$
这个公式通过指数化放大差异，并归一化确保概率和为1。

## 工作原理
它先对每个输入取指数$e^{z_i}$，突出较大值，同时抑制较小值，染回除以所有指数之和进行归一化。结果是较大的输入对应更高的概率，类似于平滑的argmax操作。


## 主要应用
在神经网络输出层，Softmax常用语图像分类、情感分析和语音识别等任务，帮助模型输出可解释的类别概率。
