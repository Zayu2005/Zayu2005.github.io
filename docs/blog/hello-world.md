## 为什么要写博客？

作为一名开发者，我一直相信「输出是最好的学习方式」。在学习新技术的过程中，我发现把知识整理成文章，不仅能加深自己的理解，也许还能帮到其他正在学习的人。

在以前，我始终使用**CSDN**记录自己的笔记，现在切换方式，使用自建博客记录自己的笔记，可以更加个性化的设计自己的网页。
!!! quote "费曼学习法"

    如果你不能用简单的语言解释一件事，说明你还没有真正理解它。

## 这个博客会写什么？

我计划围绕以下几个方向来写：

- :material-language-python: **编程语言** — 主要是 Java开发，偶尔涉及其他语言
- :material-tools: **开发工具** — 编辑器、终端、效率工具的配置和使用
- :material-server: **项目实践** — 记录个人项目的开发过程和踩坑经验
- :material-book-open-variant: **学习笔记** — 技术书籍阅读笔记、课程学习总结

## 技术选型

这个博客使用 [Zensical](https://zensical.org) 搭建，它由 Material for MkDocs 的创作者开发，具有以下优点：

- [x] 使用 Markdown 写作，简洁高效
- [x] 支持代码高亮、数学公式、图表等
- [x] 内置搜索功能
- [x] 支持亮色 / 暗色主题切换
- [x] 部署在 GitHub Pages，免费且方便
